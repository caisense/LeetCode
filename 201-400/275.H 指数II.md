# 275. H 指数 II

[https://leetcode-cn.com/problems/h-index-ii/](https://leetcode-cn.com/problems/h-index-ii/)

## 解法一：二分



```python
class Solution:
    def hIndex(self, citations) -> int:
        # 找mid，则有n-mid篇至少被引citations[mid]次
        n = len(citations)
        left = 0
        right = n - 1
        while left <= right:
            mid = left + (right - left) // 2
            if n - mid > citations[mid]:
                left = mid + 1
            else:
                right = mid - 1
        # print(left, right)
        return n - left
```
