## 图遍历dfs模板

```python
def travel(s):	#访问s节点
    #递归出口
    #前序遍历位置
    for neigh in graph[s]:  # 递归访问s的所有相邻节点，graph是邻接表
        travel(neigh)
    #后续遍历位置
```

类似于树的前序和后续遍历，图遍历的dfs也有前序和后序的方式，分别在for语句前后进行操作

### 图的前序遍历

与树的前序类似，先访问当前节点，再访问邻节点

### 图的后序遍历

当前节点的临节点全部访问完毕后，才访问当前节点
