## 一：暴力遍历（回溯）

py超时，但java没有

从0开始递归数组，直到n位置终止，每次分别回溯尝试+和-

```python
class Solution:
    def findTargetSumWays(self, nums, target: int) -> int:
        self.res = 0	#全局计数
        def bt(start, sum):
            if start  == len(nums):	#递归出口
                if sum == target:
                    self.res += 1	#满足条件，结果加一
                return
            #注意不要直接修改sum，即“保存现场”
            bt(start+1, sum+nums[start])	#加号
            bt(start+1, sum-nums[start])	#减号
        #开始
        bt(0, 0)
        return self.res
```
