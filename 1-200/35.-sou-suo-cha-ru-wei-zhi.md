# 35. 搜索插入位置

[https://leetcode-cn.com/problems/search-insert-position/](https://leetcode-cn.com/problems/search-insert-position/)

## 解法一：

一次遍历，找插入位置即可

```python
class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        i = 0
        while i < len(nums) and target > nums[i]:
            i += 1
        return i
```
